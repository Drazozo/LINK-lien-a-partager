# Link Profile

Un petit projet web pour créer une page de liens personnalisée, à la manière de Linktree.  
Affiche une photo de profil, une courte présentation et des boutons vers vos réseaux sociaux ou site web.  

Facile à personnaliser avec ton nom, ta bio et tes liens préférés.
